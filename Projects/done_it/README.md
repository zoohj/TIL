# Done-it (던잇)

> **"회의에서 업무 완료까지, AI가 던지는 프로젝트의 마침표"**

**Done-it**은 파편화된 팀 커뮤니케이션 속에서 실질적인 '실행(Action)'을 이끌어내는 AI 기반 프로젝트 매니징 에이전트입니다. <br> 단순히 대화를 요약하는 수준을 넘어, 경영학적 운영 관점을 바탕으로 팀의 의사결정을 돕고 백엔드 기술을 통해 업무 프로세스를 자동화합니다.

---

## Project Overview

* **협업의 마침표**: 대화 속에서 휘발되는 업무를 포착하여 실제 실행 자산으로 변환합니다.
* **기술과 비즈니스의 결합**: 경영학적 운영/의사결정 로직과 견고한 백엔드 시스템을 통합합니다.

---

## Core Features (MVP)

* **Context-to-Task**: 실시간 채팅 내역을 분석하여 즉시 실행 가능한 Task(할 일)로 변환 및 등록합니다.
* **Actionable Feedback**: 프로젝트 지연이나 리소스 부족 감지 시, PM의 시각에서 대안적 의사결정을 제안합니다.
* **Bridge Scheduler**: Google/Naver 캘린더와 연동하여 회의 중 결정된 일정을 자동으로 동기화합니다.

---

## System Architecture & Tech Stack

서비스의 안정성과 AI의 유연함을 동시에 확보하기 위해 **Polyglot Architecture**를 지향합니다.

| 계층 (Layer) | 역할 | 적용 기술 |
| --- | --- | --- |
| **API Gateway** | 클라이언트 요청 수신, 인증 및 라우팅 | **Java / Spring Boot** |
| **AI Service** | 자연어 이해(NLU) 및 업무 추출 로직 | **Python / FastAPI (LangChain)** |
| **Data Store** | 메시지 로그, 업무 데이터, 벡터 저장 | **PostgreSQL (pgvector)** |
| **Cache/Queue** | 실시간 알림 및 비동기 작업 처리 | **Redis** |

## Key Technical Challenges

### 1. 실행형 AI (Function Calling)

* 단순 요약을 넘어 사용자의 의도를 파악하고 외부 API를 호출합니다.
* 캘린더 등록 및 프로젝트 관리 DB 연동을 자동화합니다.

### 2. RAG 기반 히스토리 관리

* 과거의 대화 맥락을 기억하여 지능형 추론을 수행합니다.
* 벡터 DB를 활용하여 "저번 이슈"와 같은 맥락 있는 질문에 응답합니다.

### 3. 비동기 처리 및 성능 최적화

* 사용자 경험을 저해하지 않기 위해 무거운 AI 로직은 **Redis/Celery**를 활용해 비동기로 처리합니다.

---

## 🛠️ 기능 상세 명세 (Deep-Dive)

### 1. 지능형 업무 추출 및 자동 할당 (Context-to-Action)

* **기능**: 대화 내 맥락과 담당자를 매칭하여 태스크를 제안합니다.
* **기술 포인트**: Function Calling 인자를 Spring Boot `TaskService`로 전달하여 DB에 저장합니다.

### 2. 순응형 리마인드 및 팔로업 (RAG 기반)

* **기능**: 과거 데이터를 바탕으로 진행 상황을 체크하는 능동적 질문을 던집니다.
* **기술 포인트**: RAG를 활용해 과거 이력을 검색하고 현재 상태와 비교 분석합니다.

### 3. 비동기 외부 서비스 연동 (Task Automation)

* **기능**: 외부 툴(캘린더, 슬랙 등) 연동 시 발생하는 지연을 백그라운드에서 처리합니다.
* **기술 포인트**: **Redis**를 메시지 브로커로 활용하여 비동기로 외부 API를 호출합니다.

## 🐚 'Done-it' Scenario

1. **입력**: "아까 말한 테스트 코드 내일 오전까지 부탁해요" 메시지 수신.
2. **AI 분석**: FastAPI가 `{담당자, 업무, 기한}` 객체 추출.
3. **백엔드 처리**: Spring Boot가 `MemberRepository` 조회 후 `createTask()` 수행.
4. **결과**: "업무가 등록되었습니다. 캘린더에도 추가할까요?" 피드백 반환.
