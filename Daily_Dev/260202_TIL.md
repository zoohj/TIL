### 1. 백준 1654번 (랜선 자르기)
---

**이분 탐색**을 이용해 최적의 랜선 길이를 찾는 문제

* **troble shooting:** 종료 조건이 `start >= end`로 설정되어 있어, `start`와 `end`가 같아지는 마지막 정답 후보(예: 200)를 검사하지 못하고 끝나는 현상이 발생
  * 종료 조건을 `if start > end:`로 수정하여 마지막 지점까지 확인하도록 수정
* `cable_count >= target`일 때 `max_len`을 갱신하고 더 큰 범위를 찾도록

### 2. 프로그래머스 상자쌓기 문제
---

### 1. 문제의 본질

* **쌓지 않는다**
* **저장하지 않는다**
* **num이 있는 열 위에 실제로 있는 상자 개수만 센다**

---

### 2. 꼭 필요한 계산 3개

#### (1) num이 있는 층

```python
row = (num - 1) // w
```


#### (2) num이 있는 열 (방향 필수)

```python
pos = (num - 1) % w
col = pos if row % 2 == 0 else w - 1 - pos
```

#### (3) 마지막 층에 이 열이 있는지

```python
top_row = (n - 1) // w
last_cnt = n % w

if last_cnt != 0:
    if top_row % 2 == 0 and col >= last_cnt:
        top_row -= 1
    if top_row % 2 == 1 and col < w - last_cnt:
        top_row -= 1
```

---

### 3. 정답

```python
answer = top_row - row + 1
```
