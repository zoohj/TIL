## 🎧 간단 정리

### 1. 알고리즘: 좌표 이동 및 격자 탐색 (Direction Vectors)

* **핵심 원리:** `dx`, `dy` 리스트를 만들어 방향별 변화량을 저장.
```python
dx = [0, 0, 1, -1] # 하,우,상,좌 (달팽이 방향)
dy = [1, -1, 0, 0]
```
* **활용:** `nx = x + dx[i]`, `ny = y + dy[i]`를 통해 다음 좌표를 미리 계산하고, 범위 체크(`0 <= nx < n`) 및 중복 방문 확인 후 이동.

### 2. 자료구조: 원형 큐 (Circular Queue)

선형 큐의 메모리 낭비 문제를 해결하기 위해 배열의 처음과 끝을 연결한 구조.

 `⭐️ '% MAX_SIZE' 를 통한 순환`

* **나머지 연산자(`%`)의 마법:** 인덱스가 배열 끝에 도달하면 다시 0으로 돌아가게 함.
* `rear = (rear + 1) % MAX_SIZE`


* `(rear + 1) % MAX_SIZE == front`: 포화(Full) - *한 칸을 비워두는 것이 포인트*


* **주의점:** 파이썬 클래스 구현 시 **변수명과 메서드명 중복(`self.front` vs `def front()`)**은 `TypeError`를 유발, 변수명 자세하게 쓰기
