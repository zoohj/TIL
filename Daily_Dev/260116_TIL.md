## 🎧 간단 정리

### 1. 알고리즘: 좌표 이동 및 격자 탐색 (Direction Vectors)

* **핵심 원리:** `dx`, `dy` 리스트를 만들어 방향별 변화량을 저장.
```python
dx = [0, 0, 1, -1] # 하,우,상,좌 (달팽이 방향)
dy = [1, -1, 0, 0]
```
* **활용:** `nx = x + dx[i]`, `ny = y + dy[i]`를 통해 다음 좌표를 미리 계산하고, 범위 체크(`0 <= nx < n`) 및 중복 방문 확인 후 이동.

### 2. 자료구조: 원형 큐 (Circular Queue)

선형 큐의 메모리 낭비 문제를 해결하기 위해 배열의 처음과 끝을 연결한 구조.

 `⭐️ '% MAX_SIZE' 를 통한 순환`

* **나머지 연산자(`%`)의 마법:** 인덱스가 배열 끝에 도달하면 다시 0으로 돌아가게 함.
* `rear = (rear + 1) % MAX_SIZE`


* `(rear + 1) % MAX_SIZE == front`: 포화(Full) - *한 칸을 비워두는 것이 포인트*


* **주의점:** 파이썬 클래스 구현 시 **변수명과 메서드명 중복**(`self.front` vs `def front()`)은 `TypeError`를 유발, 변수명 자세하게 쓰기!

### 4. Python 데이터 정렬 (sorted & sort)

- 기본 개념
  - sorted(data): 원본은 건드리지 않고, 정렬된 새로운 리스트를 반환
  - data.sort(): 원본 리스트 자체를 정렬하며 반환값은 None
  - key 파라미터: "무엇을 기준으로 정렬할 것인가?"를 결정하는 함수를 받음

- 정렬 기준 정하기 (Key의 원리)
  - 데이터가 단순 숫자가 아니라 리스트나 딕셔너리처럼 복잡할 때, 정렬의 **'기준점'**이 필요
  - 함수 활용: 데이터의 각 요소(el)를 받아서 비교할 값만 쏙 뽑아서 리턴해주는 함수를 만듦.
  - **람다(lambda)** 활용: 이름 없는 일회용 함수로 간결하게 표현
  ```python
  data = [[1, 10], [3, 5], [2, 8]]

  # 1번 인덱스(10, 5, 8)를 기준으로 정렬하고 싶을 때
  # 방법 1: 일반 함수 사용
  def func(el):
      return el[1]
  sorted_data = sorted(data, key=func)

  # 방법 2: 람다 사용 (가장 많이 씀)
  sorted_data = sorted(data, key=lambda el: el[1])
  # 결과: [[3, 5], [2, 8], [1, 10]]
  ```
