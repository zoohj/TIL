> 1️⃣ **연결리스트**

데이터를 연속된 메모리 공간에 저장 X, 각 데이터가 다음 
데이터의 위치 가리키는 방식
Node: 데이터 저장 공간, 다음 노드 참조(주소)

**특징**: 동적 크기 조절, 삽입&삭제 쉬움, 메모리 효율성(배열은 미리 메모리 할당) 




```python
class Node:
  def __init__(self, data):
    self.data = data
    self.next = None

head = Node(1)
head.next = Node(2)
head.next.next=Node(3)
```

> 2️⃣ **Stack**
- 메소드
  - **push(x)** : 데이터 x를 스택에 넣음
  - **pop()** : 스택에서 데이터 꺼냄
  - **peek()** : 마지막 데이터 확인 (제거 X)
  - **is_empty()** : 스택 비어있는지 확인


```python
class Node:
  def __init__(self, data):
    self.data = data
    self.next = None

class Stack:
  def __init__(self):
    self.top = None

  def push(self, data):
    if self.top is None:
      self.top = Node(data)
    else:
      node = Node(data)
      node.next = self.top
      self.top = node
  
  def pop(self):
    if self.top is None:
      return None
    node = self.top
    self.top = self.top.next
    return node.data
  
  def peek(self):
    if self.top is None:
      return None
    node = self.top
    return node.data
  
  def is_empty(self):
    return self.top is None 
    # self.top is None => True/False


if __name__ == "__main__":
  s = Stack()

  for i in range(3):
    data = chr(ord("A")+i)
    s.push(data)
    print(f"Push data = {s.peek()})")
  print()

  while not s.is_empty():
    print(f"Pop data = {s.pop()}")

  print()

  print(f"Peek data = {s.peek()})")


```
