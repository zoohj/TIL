> **리스트 컴프리헨션**
```python
# 행렬 초기화
result = [[0] * M for _ in range(N)]
```
> **행렬(2차원 리스트) 출력**
```python
# 기존 코드
for row in result:
    for val in row:
        print(val, end=" ")
    print()

# 개선 코드(언팩킹)
for row in result:
    print(*row)
```
