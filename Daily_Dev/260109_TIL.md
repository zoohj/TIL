## 🎧 간단 정리

### 1. 파이썬 기본 문법 및 tip 💡

* **입력 처리:** `input().strip()`은 문자열로 저장됨. 정수 계산 시 반드시 `int()` 형변환 필수.
* **문자열 뒤집기:** `reversed_n = str_n[::-1]` (슬라이싱 활용).
* **리스트 vs 튜플:** 
  * `[ ]` **리스트**: 수정 가능(Mutable), `append()`, `pop()` 사용.
  * `( )` **튜플**: 수정 불가(Immutable), 정렬 기준 설정(`key=lambda`)이나 `set`의 요소로 쓰기 좋음.


* **`for-else` 문:** 반복문이 `break` 없이 정상적으로 끝났을 때만 `else` 블록이 실행됨. (찾는 값이 없을 때 예외 처리 용도)

### 2. 슬라이딩 윈도우 (Sliding Window) 윈도우를 미는 기술
> **슬라이싱 방식:** 
* `array[i : i + win]`으로 직접 잘라내어 계산. 코드가 직관적이지만 윈도우가 크면 느려질 수 있음.
>**갱신 방식:**
*  `이전 합 - 맨 앞 숫자(i-1) + 새 숫자(i+win-1)`. 연산 속도가 매우 빠름 ().
>**인덱스 공식:**
*  시작점의 범위 -  **`range(len(numbers) - win + 1)`**.

---

### 3. 정렬 및 데이터 처리 (Sorting)

* **데이터 타입 주의:** 문자열 상태로 정렬하면 사전순(`'10' < '2'`)으로 정렬됨. 수 정렬 시 반드시 `int` 변환.
* **중복 제거:** `set()`을 통해 순식간에 중복 제거 가능..

* **최빈값과 Offset:** 음수 인덱스를 대신하기 위해(lst[0]부터 시작) `+1000` 같은 **Offset**을 더해 양수 인덱스로 변환 후 사용. 

---

### 4. 자료구조: 스택(Stack) & 연결 리스트

#### **연결 리스트 구현 주의점:**
> **연결 고리 유지:**  

`push` 시 `새 노드.next = top`을 먼저 하고, 그 다음 `top = 새 노드`를 지정해야 데이터가 안 끊김. 

> **이름 충돌:** 

멤버 변수 `self.top`과 메서드 `top()`의 이름이 같으면 안 됨 → `self._top` 추천.

> **순회(Size)** 

`while node: node = node.next`를 사용하여 끝(`None`)까지 이동.

> **효율성:**

`size`를 매번 세지 말고, `self._size` 변수를 만들어 `push/pop` 때마다 증감시키면 $O(1)$로 최적화됨.
