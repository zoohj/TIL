### 파이썬 기초 개념
> **문법**
- `( )` **tuple**: `tup = (a, b, c)`
  - 변경 불가능한 list

- `{ }` **Set**: `set_ex = {a, b, c, d}`
  - 순서 없이 저장, 중복 허용 X
  - 집합의 원소: 기본 자료형, 변경 불가능한 컨테이너(`tuple, range` etc)
  - 변경 가능한 컨테이너에는 `리스트, 집합, 딕셔너리` 허용 X
  - set method
    - 
    - `set_ex.add(원소)`: 집합 원소 삽입
    - `set_ex.update(컨테이너)`
      ```python 
      numbers.update([4, 5, 6])  # numbers에 리스트 [4, 5, 6]의 원소들을 삽입
      ```
    - `set_ex.remove()`, `set_ex.discard()`
      - remove: 원소 존재 X => 에러 발생
      - discard: 원소 존재 X => 에러 발생 X


- `{ }` **딕셔너리**: `dic = {"key": "value"}` 
  - 키로 접근하고 키값을 통해 value값을 알 수 있음
    ```python
      dic, dic.keys    # key
      dic[key]         #value
    ```

  - dictonary method
    - 
    - `dic.get("key")` => value 반환
    - `dic.pop("key")` => 키쌍 제거 및 value 반환
    - `dic.keys()` => 모든 키 목록 
    - `dic.values()` => 모든 밸류 목록
    - `dic.items()` => 모든 key-value 쌍 목록
    - `dic["key"]="value"` => key와 value 다 넣음

```python
empty_dict = {}       # 딕셔너리
empty_dict = set()    # 집합
```

  - `array[][]` **이차원 배열**
    - 좌표 쓸 때 사용
    - `arr[i][j]` i행, j열  
    - 이차원 배열 선언 `arr = [[0] * j for _ in range(i)]`
    - **경계처리 실수 주의**
