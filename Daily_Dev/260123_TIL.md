### Database(PostgreSQL)
> **NULL** 데이터 검색 

데이터베이스에서 값이 비어 있는 상태(NULL)를 찾을 때는 <br>
'=' 연산자가 아닌 ``IS NULL``을 사용
```sql
SELECT * FROM _ WHERE _ IS NULL 
                        IS NOT NULL 
```

### 알고리즘

#### 1. 파도반 수열 (백준 9461)
* **알고리즘:** 다이나믹 프로그래밍 (DP) - 규칙성 찾아 점화식 도출
* **주의사항:** N이 작은 경우(1~5) `IndexError`가 발생하지 않도록 초기값 설정 시 배열 크기를 넉넉히(예: `max(num, 5) + 1`) 잡는 것이 중요함

#### 2. 구간 합 구하기 4 (백준 11659)
* **알고리즘:** 누적 합 (Prefix Sum)
* **핵심:** 반복되는 구간 합 계산을 $O(N \times M)$에서 $O(N + M)$으로 최적화.
* **인덱스 설계:**
  * `prefix_sum[0] = 0`으로 설정하여 계산의 편의성 확보.
  * `a`번째부터 ``b``번째까지의 합 공식: `prefix_sum[b] - prefix_sum[a-1]`
  * 파이썬 리스트(`input_data`)는 0부터 시작하지만, <br>
  누적 합 배열은 문제의 순서(1번째~)와 인덱스를 맞추기 위해 하나 더 크게 생성함.
